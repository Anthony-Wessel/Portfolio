10 Years of Experience

I've been programming since I was a freshman in high school in 2014. I was learning Java for my school's FIRST Robotics team, and I spent a lot of my free time using online tutorials to learn how to make games. I've come a long way since then, learning several new languages, engines, and approaches that have improved my code and problem solving.

FIRST Robotics

I spent 4 years on my high school FIRST Robotics team and gained a lot of experience. While a lot of the code was simple, I got the chance to work on some complex systems as well, such as computer vision, a mathematically complex swerve drive, and a custom scripting solution for designing autonomous routines.

Physical Projects

I've enjoyed using my programming experience outside of games as well. I have a 3D printer and some electronics experience, which I've used to make some pretty cool stuff. My personal favorite is a lamp that I designed from scratch. I designed and 3D printed the model, soldered the electronics together, and programmed a microcontroller to run the 300 RGB LEDs. Using so many LEDs meant I had to focus heavily on optimizations to keep the animation of the lights smooth.
